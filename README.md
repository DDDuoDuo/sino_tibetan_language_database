## 项目简介 | Introduction

汉藏语分化时间数据库旨在通过历史比较法，基于词汇证据，收集汉藏语系内各语言与方言的分化时间信息。

这些词汇证据自带时间信息，如工具材料名称（石、铁、铜）、驯化动物、外来物种或概念、借词等。结合考古研究结果和历史文献资料，通过将词汇进行**历史分层（Lexical Stratigraphic Analysis）**，可以判定目标语言或方言的分化上下限。用户可通过本数据库追溯语言分化历程，并比较语系内不同分支的分化模式。
